# Remove-AzAksArcNodepool

## SYNOPSIS
Deletes the specified agent pool in the provisioned cluster

## SYNTAX

### Delete (Default)
```
Remove-AzAksArcNodepool -ClusterName <String> -ResourceGroupName <String> -Name <String>
 [-SubscriptionId <String>] [-AsJob] [-NoWait] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzAksArcNodepool -ClusterName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 -InputObject <IAksArcIdentity> [-AsJob] [-NoWait] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Deletes the specified agent pool in the provisioned cluster

## EXAMPLES

### Example 1: Remove a nodepool in a provisioned cluster.
```powershell
Remove-AzAksArcNodepool -ClusterName azps_test_cluster -ResourceGroupName azps_test_group -Name azps_test_nodepool_example
```

Deletes the nodepool in the provisioned cluster.
