# Stop-AzVM

## SYNOPSIS
The operation to power off (stop) a virtual machine.
The virtual machine can be restarted with the same provisioned resources.
You are still charged for this virtual machine.

## SYNTAX

### PowerOff (Default)
```
Stop-AzVM -Name <String> -ResourceGroupName <String> -SubscriptionId <String> [-DefaultProfile <PSObject>]
 [-AsJob] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### PowerOffViaIdentity
```
Stop-AzVM -InputObject <IComputeIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
The operation to power off (stop) a virtual machine.
The virtual machine can be restarted with the same provisioned resources.
You are still charged for this virtual machine.
