## SYNTAX

```
Get-AzVMImagePublisher -Location <String> -SubscriptionId <String[]> [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## DESCRIPTION
Gets a list of virtual machine image publishers for the specified Azure location.
