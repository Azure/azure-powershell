# Get-AzDiskEncryptionSet

## SYNOPSIS
Get or list disk encryption sets.

## SYNTAX

```
Get-AzDiskEncryptionSet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Get or list disk encryption sets.
