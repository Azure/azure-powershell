# Get-AzRecoveryServicesBackupWorkloadItem

## SYNOPSIS
Provides a pageable list of workload item of a specific container according to the query filter and the pagination\r\nparameters.

## SYNTAX

```
Get-AzRecoveryServicesBackupWorkloadItem -ContainerName <String> -FabricName <String>
 -ResourceGroupName <String> -VaultName <String> [-SubscriptionId <String[]>] [-Filter <String>]
 [-SkipToken <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Provides a pageable list of workload item of a specific container according to the query filter and the pagination\r\nparameters.
