# Get-AzRecoveryServicesTargetComputeSize

## SYNOPSIS
Lists the available target compute sizes for a replication protected item.

## SYNTAX

```
Get-AzRecoveryServicesTargetComputeSize -FabricName <String> -ProtectionContainerName <String>
 -ReplicatedProtectedItemName <String> -ResourceGroupName <String> -ResourceName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Lists the available target compute sizes for a replication protected item.
