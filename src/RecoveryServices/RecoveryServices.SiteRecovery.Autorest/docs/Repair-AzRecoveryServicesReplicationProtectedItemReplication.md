# Repair-AzRecoveryServicesReplicationProtectedItemReplication

## SYNOPSIS
The operation to start resynchronize/repair replication for a replication protected item requiring resynchronization.

## SYNTAX

```
Repair-AzRecoveryServicesReplicationProtectedItemReplication -FabricName <String>
 -ProtectionContainerName <String> -ReplicatedProtectedItemName <String> -ResourceGroupName <String>
 -ResourceName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
The operation to start resynchronize/repair replication for a replication protected item requiring resynchronization.
