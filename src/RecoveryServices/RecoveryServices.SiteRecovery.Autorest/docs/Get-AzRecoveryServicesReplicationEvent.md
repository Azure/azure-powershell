# Get-AzRecoveryServicesReplicationEvent

## SYNOPSIS
The operation to get the details of an Azure Site recovery event.

## SYNTAX

### List (Default)
```
Get-AzRecoveryServicesReplicationEvent -ResourceGroupName <String> -ResourceName <String>
 [-SubscriptionId <String[]>] [-Filter <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Get
```
Get-AzRecoveryServicesReplicationEvent -EventName <String> -ResourceGroupName <String> -ResourceName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```
