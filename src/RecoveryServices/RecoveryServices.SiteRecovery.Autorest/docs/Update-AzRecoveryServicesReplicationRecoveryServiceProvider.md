# Update-AzRecoveryServicesReplicationRecoveryServiceProvider

## SYNOPSIS

### Refresh (Default)
```
Update-AzRecoveryServicesReplicationRecoveryServiceProvider -FabricName <String> -ProviderName <String>
 -ResourceGroupName <String> -ResourceName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>]

```
