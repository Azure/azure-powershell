# Add-AzRecoveryServicesReplicationProtectedItemDisk

## SYNOPSIS
Operation to add disks(s) to the replication protected item.

## SYNTAX

### AddExpanded (Default)
```
Add-AzRecoveryServicesReplicationProtectedItemDisk -FabricName <String> -ProtectionContainerName <String>
 -ReplicatedProtectedItemName <String> -ResourceGroupName <String> -ResourceName <String>
 [-SubscriptionId <String>] [-ProviderSpecificDetailInstanceType <String>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Add
```
Add-AzRecoveryServicesReplicationProtectedItemDisk -FabricName <String> -ProtectionContainerName <String>
 -ReplicatedProtectedItemName <String> -ResourceGroupName <String> -ResourceName <String>
 -AddDisksInput <IAddDisksInput> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```
