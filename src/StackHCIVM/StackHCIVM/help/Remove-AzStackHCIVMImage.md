# Remove-AzStackHCIVMImage

## SYNOPSIS
The operation to delete an image.

## SYNTAX

### ByResourceId (Default)
```
Remove-AzStackHCIVMImage [-SubscriptionId <String>] -ResourceId <String> [-Force] [-Break]
 [-HttpPipelineAppend <SendAsyncStep[]>] [-HttpPipelinePrepend <SendAsyncStep[]>] [-NoWait] [-Proxy <Uri>]
 [-ProxyCredential <PSCredential>] [-ProxyUseDefaultCredentials] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Remove-AzStackHCIVMImage -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>] [-Force]
 [-Break] [-HttpPipelineAppend <SendAsyncStep[]>] [-HttpPipelinePrepend <SendAsyncStep[]>] [-NoWait]
 [-Proxy <Uri>] [-ProxyCredential <PSCredential>] [-ProxyUseDefaultCredentials] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
The operation to delete an image.

## EXAMPLES

### EXAMPLE 1
```
Remove-AzStackHCIVMImage -Name  "testImage" -ResourceGroupName "test-rg"
```
