# Get-AzRelayOperation

## SYNOPSIS
List supported Relay Operations

## SYNTAX

```
Get-AzRelayOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
The **Get-AzRelayOperation** cmdlet Lists the Relay supported Operations.
