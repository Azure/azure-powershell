# Set-AzServiceBusNamespaceIPFilterRule

## SYNOPSIS
Creates or updates an IpFilterRule for a Namespace.

## SYNTAX

### UpdateExpanded (Default)
```
Set-AzServiceBusNamespaceIPFilterRule -IPFilterRuleName <String> -NamespaceName <String>
 -ResourceGroupName <String> -SubscriptionId <String> [-Action <IPAction>] [-FilterName <String>]
 [-IPMask <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Set-AzServiceBusNamespaceIPFilterRule -InputObject <IServiceBusIdentity> [-Action <IPAction>]
 [-FilterName <String>] [-IPMask <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### UpdateViaIdentity
```
Set-AzServiceBusNamespaceIPFilterRule -InputObject <IServiceBusIdentity> [-Parameter <IIPFilterRule>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates or updates an IpFilterRule for a Namespace.
