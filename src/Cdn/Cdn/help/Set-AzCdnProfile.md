# Set-AzCdnProfile

## SYNOPSIS
Updates a CDN profile.

## SYNTAX

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
The **Set-AzCdnProfile** cmdlet updates an Azure Content Delivery Network (CDN) profile.
