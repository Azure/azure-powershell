### Example 1: List CDN WAF policies in a resource group
```powershell
Get-AzCdnPolicy -ResourceGroupName testps-rg-da16jm
```

Lists CDN WAF policies in the specified resource group.

### Example 2: Get a CDN WAF policy
```powershell
Get-AzCdnPolicy -ResourceGroupName testps-rg-da16jm -Name policy001
```

Gets the specified CDN WAF policy.
