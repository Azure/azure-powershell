# New-AzCdnProfileAgent

## SYNOPSIS
Create an agent link (web agent association) within a CDN profile.

## SYNTAX

### CreateExpanded (Default)
```
New-AzCdnProfileAgent -AgentName <String> -ProfileName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-CustomDomain <IResourceReference[]>] [-WebAgentId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentityProfile
```
New-AzCdnProfileAgent -AgentName <String> -ProfileInputObject <ICdnIdentity> -ProfileAgent <IProfileAgent>
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentityProfileExpanded
```
New-AzCdnProfileAgent -AgentName <String> -ProfileInputObject <ICdnIdentity>
 [-CustomDomain <IResourceReference[]>] [-WebAgentId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaJsonFilePath
```
New-AzCdnProfileAgent -AgentName <String> -ProfileName <String> -ResourceGroupName <String>
 -JsonFilePath <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### CreateViaJsonString
```
New-AzCdnProfileAgent -AgentName <String> -ProfileName <String> -ResourceGroupName <String>
 -JsonString <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create an agent link (web agent association) within a CDN profile.
