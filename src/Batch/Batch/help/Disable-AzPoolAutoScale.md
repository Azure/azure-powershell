# Disable-AzPoolAutoScale

## SYNOPSIS
Disables automatic scaling for a pool.

## SYNTAX

### Disable (Default)
```
Disable-AzPoolAutoScale -AccountName <String> -PoolName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DisableViaIdentityBatchAccount
```
Disable-AzPoolAutoScale -PoolName <String> -BatchAccountInputObject <IBatchIdentity>
 [-DefaultProfile <PSObject>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DisableViaIdentity
```
Disable-AzPoolAutoScale -InputObject <IBatchIdentity> [-DefaultProfile <PSObject>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Disables automatic scaling for a pool.
