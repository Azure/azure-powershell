# Update-AzServiceEndpointPolicyDefinition

## SYNOPSIS
Incrementally updates service endpoint policy definition of a service endpoint policy.

## SYNTAX

```
Update-AzServiceEndpointPolicyDefinition -Name <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-Description <String>] [-ServiceResource <String[]>] [-Service <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Update-AzServiceEndpointPolicyDefinition** incrementally updates service endpoint policy definition of a service endpoint policy. I.e. only the specified parameters values are changed and values of the unspecified properties are kept unlike of Set-AzServiceEndpointPolicyDefinition cmdlet.

## EXAMPLES

### Example 1
```powershell
PS C:> Update-AzServiceEndpointPolicyDefinition -ServiceEndpointPolicy $serviceEndpointPolicy -Name $serviceEndpointPolicyDefinitionName -Description $serviceEndpointPolicyDefinitionDescription;
```

This command updates description in the service endpoint policy definition keeping values of the other properties the same.
