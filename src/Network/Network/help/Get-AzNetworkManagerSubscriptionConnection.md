# Get-AzNetworkManagerSubscriptionConnection

## SYNOPSIS
Gets a network manager subscription connection.

## SYNTAX

### NoExpand (Default)
```
Get-AzNetworkManagerSubscriptionConnection [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Expand
```
Get-AzNetworkManagerSubscriptionConnection -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```
