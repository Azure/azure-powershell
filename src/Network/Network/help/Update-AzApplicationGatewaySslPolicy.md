# Update-AzApplicationGatewaySslPolicy

## SYNOPSIS
Incrementally updates SSL policy of an application gateway.

## SYNTAX

```
Update-AzApplicationGatewaySslPolicy -ApplicationGateway <PSApplicationGateway>
 [-DisabledSslProtocols <String[]>] [-PolicyType <String>] [-PolicyName <String>] [-CipherSuite <String[]>]
 [-MinProtocolVersion <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
The **Update-AzApplicationGatewaySslPolicy** incrementally updates SSL policy of an application gateway. I.e. only the specified parameters values are changed and values of the unspecified properties are kept unlike of Set-AzApplicationGatewaySslPolicy cmdlet.

## EXAMPLES

### Example 1
```powershell
PS C:> Update-AzApplicationGatewaySslPolicy -ApplicationGateway $appgw -PolicyName AppGwSslPolicy20150501
```

This command updates SSL policy name of application gateway keeping other properties unchanged.
