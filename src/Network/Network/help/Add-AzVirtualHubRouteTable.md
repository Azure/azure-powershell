# Add-AzVirtualHubRouteTable

## SYNOPSIS
Creates a Virtual Hub Route Table resource which is a child of VirtualHub.

## SYNTAX

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
The virtual hub route table resource contains the list of routes and a list of connections to which it can 
be attached to and is used to route traffic in a Virtual Hub.
