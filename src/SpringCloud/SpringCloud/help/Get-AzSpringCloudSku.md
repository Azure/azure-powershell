# Get-AzSpringCloudSku

## SYNOPSIS
Lists all of the available skus of the Microsoft.AppPlatform provider.

## SYNTAX

```
Get-AzSpringCloudSku [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Lists all of the available skus of the Microsoft.AppPlatform provider.
