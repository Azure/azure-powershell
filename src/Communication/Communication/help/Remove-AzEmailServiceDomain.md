# Remove-AzEmailServiceDomain

## SYNOPSIS
Operation to delete a Domains resource.

## SYNTAX

### Delete (Default)
```
Remove-AzEmailServiceDomain -EmailServiceName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentityEmailService
```
Remove-AzEmailServiceDomain -Name <String> -EmailServiceInputObject <IEmailServiceIdentity>
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzEmailServiceDomain -InputObject <IEmailServiceIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Operation to delete a Domains resource.

## EXAMPLES

### Example 1: Removes Email service custom domain.
```powershell
Remove-AzEmailServiceDomain -Name test.net -EmailServiceName ContosoAcsResource1 -ResourceGroupName ContosoResourceProvider1
```

Removes Email service custom domain.

### Example 2: Removes Email service azure managed domain.
```powershell
Remove-AzEmailServiceDomain -Name AzureManagedDomain -EmailServiceName ContosoAcsResource1 -ResourceGroupName ContosoResourceProvider1
```

Removes Email service azure managed domain.
