# Remove-AzQuotaGroupQuota

## SYNOPSIS
Deletes the GroupQuotas for the name passed.
All the remaining shareQuota in the GroupQuotas will be lost.

## SYNTAX

### Delete (Default)
```
Remove-AzQuotaGroupQuota -ManagementGroupId <String> -Name <String> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzQuotaGroupQuota -InputObject <IQuotaIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentityManagementGroup
```
Remove-AzQuotaGroupQuota -ManagementGroupInputObject <IQuotaIdentity> -Name <String>
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Deletes the GroupQuotas for the name passed.
All the remaining shareQuota in the GroupQuotas will be lost.
