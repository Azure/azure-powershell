# Remove-AzArcGateway

## SYNOPSIS
The operation to delete a gateway.

## SYNTAX

### Delete (Default)
```
Remove-AzArcGateway -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzArcGateway -InputObject <IArcGatewayIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The operation to delete a gateway.

## EXAMPLES

### Example 1: Remove a gateway for arc machine
```powershell
Remove-AzArcGateway -Name "myArcGateway" -ResourceGroupName "ytongtest"
```

Remove a gateway for arc machine
