# New-AzAutoscaleWebhookNotificationObject

## SYNOPSIS
Create an in-memory object for WebhookNotification.

## SYNTAX

```
New-AzAutoscaleWebhookNotificationObject [-Property <IWebhookNotificationProperties>] [-ServiceUri <String>]
 [<CommonParameters>]
```

## DESCRIPTION
Create an in-memory object for WebhookNotification.

## EXAMPLES
