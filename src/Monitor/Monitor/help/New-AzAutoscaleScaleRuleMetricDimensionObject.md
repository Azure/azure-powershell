# New-AzAutoscaleScaleRuleMetricDimensionObject

## SYNOPSIS
Create an in-memory object for ScaleRuleMetricDimension.

## SYNTAX

```

 -Operator <ScaleRuleMetricDimensionOperationType> -Value <String[]> [<CommonParameters>]
```
