# New-AzScheduledQueryRuleLogMetricTrigger

## SYNOPSIS
Creates an object of type Log Metric Trigger

## SYNTAX

```
New-AzScheduledQueryRuleLogMetricTrigger -ThresholdOperator <String> -Threshold <Double>
 -MetricTriggerType <String> -MetricColumn <String> [<CommonParameters>]
```

## DESCRIPTION
Creates an object of type Log Metric Trigger and is optional.
This is the trigger condition for metric query rule, to be used when you need to state metric measurement type of alert.

## EXAMPLES

### Example 1
```powershell
PS C:\> $metricTrigger = New-AzScheduledQueryRuleLogMetricTrigger -ThresholdOperator "GreaterThan" -Threshold 5 -MetricTriggerType "Consecutive" -MetricColumn "Computer"
```
