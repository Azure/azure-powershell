# Get-AzLogProfile

## SYNOPSIS
Gets a log profile.

## SYNTAX

```
Get-AzLogProfile [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
The **Get-AzLogProfile** cmdlet gets a log profile.

## EXAMPLES
### Example 1: Gets a log profile

Get-AzLogProfile
