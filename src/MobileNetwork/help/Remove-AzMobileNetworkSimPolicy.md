# Remove-AzMobileNetworkSimPolicy

## SYNOPSIS
Deletes the specified SIM policy.

## SYNTAX

### Delete (Default)
```
Remove-AzMobileNetworkSimPolicy -MobileNetworkName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzMobileNetworkSimPolicy -InputObject <IMobileNetworkIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Deletes the specified SIM policy.

## EXAMPLES

### Example 1: Deletes the specified SIM policy.
```powershell
Remove-AzMobileNetworkSimPolicy -MobileNetworkName azps-mn -Name azps-mn-simpolicy -ResourceGroupName azps_test_group
```

Deletes the specified SIM policy.
