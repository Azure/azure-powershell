### Example 1: Get the AuthKeys for a given Sql Migration Service
```powershell
PS C:\> Get-AzDataMigrationSqlServiceAuthKey -ResourceGroupName "MyResourceGroup" -SqlMigrationServiceName "MySqlMigrationService"
```
