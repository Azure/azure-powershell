# Remove-AzScVmmVMExtension

## SYNOPSIS
The operation to remove the extension on a virtual machine.

## SYNTAX

```
Remove-AzScVmmVMExtension -ExtensionName <String> -ResourceGroupName <String> -vmName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
The operation to remove the extension on a virtual machine.

## EXAMPLES

### Example 1: Remove extension of a SCVMM VM
```powershell
Remove-AzScVmmVMExtension -ExtensionName 'RunCommand' -vmName 'test-vm' -ResourceGroupName 'test-rg-01'
```

Remove extension of a SCVMM VM
