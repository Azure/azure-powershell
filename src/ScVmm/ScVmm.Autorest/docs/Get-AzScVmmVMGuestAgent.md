# Get-AzScVmmVMGuestAgent

## SYNOPSIS
Implements GuestAgent GET method.

## SYNTAX

```
Get-AzScVmmVMGuestAgent -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Implements GuestAgent GET method.
