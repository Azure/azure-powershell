# Set-AzScVmmVirtualMachineTemplate

## SYNOPSIS
Onboards the ScVmm VM Template as an Azure VM Template resource.

## SYNTAX

### UpdateExpanded (Default)
```
Set-AzScVmmVirtualMachineTemplate -Name <String> -ResourceGroupName <String> -Location <String>
 [-SubscriptionId <String>] [-ExtendedLocationName <String>] [-ExtendedLocationType <String>]
 [-InventoryItemId <String>] [-Tag <Hashtable>] [-Uuid <String>] [-VmmServerId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaJsonFilePath
```
Set-AzScVmmVirtualMachineTemplate -Name <String> -ResourceGroupName <String> -JsonFilePath <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### UpdateViaJsonString
```
Set-AzScVmmVirtualMachineTemplate -Name <String> -ResourceGroupName <String> -JsonString <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Onboards the ScVmm VM Template as an Azure VM Template resource.
