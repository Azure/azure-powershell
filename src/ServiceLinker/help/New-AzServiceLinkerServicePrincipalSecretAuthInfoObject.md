# New-AzServiceLinkerServicePrincipalSecretAuthInfoObject

## SYNOPSIS

## SYNTAX

```
New-AzServiceLinkerServicePrincipalSecretAuthInfoObject -ClientId <String> -PrincipalId <String>
 -Secret <String> [<CommonParameters>]
```

## DESCRIPTION
