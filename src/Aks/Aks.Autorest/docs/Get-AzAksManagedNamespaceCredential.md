# Get-AzAksManagedNamespaceCredential

## SYNOPSIS

## SYNTAX

```
Get-AzAksManagedNamespaceCredential -ManagedNamespaceName <String> -ResourceGroupName <String>
 -ResourceName <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Lists the credentials of a namespace.
