# Stop-AzAksDashboard

## SYNOPSIS
Stop the Kubectl SSH tunnel created in Start-AzKubernetesDashboard.

## SYNTAX

```
Stop-AzAksDashboard [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-SubscriptionId <String>] [<CommonParameters>]
```

## DESCRIPTION
Stop the Kubectl SSH tunnel created in Start-AzKubernetesDashboard.

## EXAMPLES

### Example 1
```powershell
Stop-AzAksDashboard
```

Stops the existing SSH tunnel setup by executing Start-AzKubernetesDashboard.
