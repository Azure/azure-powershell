# Set-AzAksCluster

## SYNOPSIS
Update or create a managed Kubernetes cluster.

## DESCRIPTION
Update or create a managed Kubernetes cluster.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-AzAks -ResourceGroupName group -Name myCluster | Set-AzAks -NodeCount 5
```

Set the number of nodes in the Kubernetes cluster to 5.
