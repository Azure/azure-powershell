# New-AzADB2CTenant

## SYNOPSIS
Initiates an async request to create both the Azure AD B2C tenant and the corresponding Azure resource linked to a subscription.

## SYNTAX

```
New-AzADB2CTenant -Name <String> -ResourceGroupName <String> -CountryCode <String> -DisplayName <String>
 -Location <String> -Sku <B2CResourceSkuname> [-SubscriptionId <String>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Initiates an async request to create both the Azure AD B2C tenant and the corresponding Azure resource linked to a subscription.
