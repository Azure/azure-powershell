# Set-AzEventHubNamespace

## SYNOPSIS
Updates the specified Event Hubs namespace.

## SYNTAX

### NamespaceParameterSet (Default)

## DESCRIPTION
The Set-AzEventHubNamespace cmdlet updates the properties of the specified Event Hubs namespace.
