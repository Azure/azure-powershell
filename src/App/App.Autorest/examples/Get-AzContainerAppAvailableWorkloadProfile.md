### Example 1: Get all available workload profiles for a location.
```powershell
Get-AzContainerAppAvailableWorkloadProfile -Location eastus
```

```output
Name        Location
----        --------
D4          eastus
D8          eastus
D16         eastus
D32         eastus
E4          eastus
E8          eastus
E16         eastus
E32         eastus
Consumption eastus
```

Get all available workload profiles for a location.
