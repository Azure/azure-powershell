# New-AzAppIdentity

## SYNOPSIS
Create an in-memory object for AppIdentity.

## SYNTAX

```
New-AzAppIdentity [-AuthConfigName <String>] [-CertificateName <String>] [-ComponentName <String>]
 [-ContainerAppName <String>] [-EnvironmentName <String>] [-Id <String>] [-ReplicaName <String>]
 [-ResourceGroupName <String>] [-RevisionName <String>] [-SourceControlName <String>] [-StorageName <String>]
 [-SubscriptionId <String>] [<CommonParameters>]
```

## DESCRIPTION
