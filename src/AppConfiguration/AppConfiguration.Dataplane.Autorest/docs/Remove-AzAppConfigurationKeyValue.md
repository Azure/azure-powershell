# Remove-AzAppConfigurationKeyValue

## SYNOPSIS
Deletes a key-value.

## SYNTAX

### Delete (Default)
```
Remove-AzAppConfigurationKeyValue -Endpoint <String> -Key <String> [-Label <String>] [-IfMatch <String>]
 [-SyncToken <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzAppConfigurationKeyValue -Endpoint <String> -InputObject <IAppConfigurationIdentity>
 [-Label <String>] [-IfMatch <String>] [-SyncToken <String>] [-DefaultProfile <PSObject>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Deletes a key-value.
