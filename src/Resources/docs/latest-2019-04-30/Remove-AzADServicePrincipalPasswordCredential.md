# Remove-AzADServicePrincipalPasswordCredential

## SYNOPSIS
Removes a password credential associated with an service principal.

## SYNTAX

```
Remove-AzADServicePrincipalPasswordCredential -ObjectId <String> -TenantId <String> [-KeyId <String>]
 [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Removes a password credential associated with an service principal.
