# New-AzADApplicationPasswordCredential

## SYNOPSIS
Adds a new password credential to an application.

## SYNTAX

```
New-AzADApplicationPasswordCredential -ObjectId <String> -TenantId <String> [-EndDate <DateTime>]
 [-Password <SecureString>] [-StartDate <DateTime>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Adds a new password credential to an application.
