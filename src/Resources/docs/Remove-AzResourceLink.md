# Remove-AzResourceLink

## SYNOPSIS
Deletes a resource link with the specified ID.

## SYNTAX

### Delete (Default)
```
Remove-AzResourceLink -ResourceId <String> [-PassThru] [-DefaultProfile <PSObject>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzResourceLink -InputObject <IResourcesIdentity> [-PassThru] [-DefaultProfile <PSObject>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Deletes a resource link with the specified ID.
