# Get-AzMgUser

## SYNOPSIS
Represents an Azure Active Directory user object.

## SYNTAX

### List (Default)
```
Get-AzMgUser [-Count] [-Expand <String[]>] [-Filter <String>] [-Orderby <String[]>] [-Search <String>]
 [-Select <String[]>] [-Skip <Int32>] [-Top <Int32>] [-ConsistencyLevel <String>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

### Get
```
Get-AzMgUser -Id <String> [-Expand <String[]>] [-Select <String[]>] [-ConsistencyLevel <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Represents an Azure Active Directory user object.
