# Get-AzsOffer

## SYNOPSIS
Get the list of public offers.

## SYNTAX

```
Get-AzsOffer [[-Skip] <Int32>] [[-Top] <Int32>] [[-Provider] <String>] [<CommonParameters>]
```

## DESCRIPTION
Get the list of public offers.

## EXAMPLES

### EXAMPLE 1

```
Get-AzsOffer | fl
```
