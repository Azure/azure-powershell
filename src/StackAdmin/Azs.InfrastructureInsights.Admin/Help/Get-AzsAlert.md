# Get-AzsAlert

## SYNOPSIS
Returns alerts at a given location.

## SYNTAX

### Alerts_List (Default)
```
Get-AzsAlert [-Filter <String>] [-Skip <Int32>] -Location <String> [-Top <Int32>] [<CommonParameters>]
```

### Alerts_Get
```
Get-AzsAlert -Location <String> -AlertName <String> [<CommonParameters>]
```

## DESCRIPTION
Returns alerts at a given location.
