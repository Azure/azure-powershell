# Start-AzsReclaimStorageCapacity

## SYNOPSIS

## SYNTAX

```
Start-AzsReclaimStorageCapacity [-Location <String>] [-AsJob] [-Force] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Start garbage collection on deleted storage objects.

## EXAMPLES

### EXAMPLE 1
```
Start-AzsReclaimStorageCapacity
```

Start garbage collection.
