# Get-AzMapsSubscriptionOperation

## SYNOPSIS
List operations available for the Maps Resource Provider

## SYNTAX

```
Get-AzMapsSubscriptionOperation [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
List operations available for the Maps Resource Provider
