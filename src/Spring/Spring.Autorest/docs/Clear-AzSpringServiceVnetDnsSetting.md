# Clear-AzSpringServiceVnetDnsSetting

## SYNOPSIS
Flush Virtual Network DNS settings for a VNET injected Service.

## SYNTAX

### Flush (Default)
```
Clear-AzSpringServiceVnetDnsSetting -ResourceGroupName <String> -ServiceName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### FlushViaIdentity
```
Clear-AzSpringServiceVnetDnsSetting -InputObject <ISpringAppsIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Flush Virtual Network DNS settings for a VNET injected Service.
