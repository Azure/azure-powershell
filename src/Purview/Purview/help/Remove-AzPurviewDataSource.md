# Remove-AzPurviewDataSource

## SYNOPSIS
Deletes a data source

## SYNTAX

```
Remove-AzPurviewDataSource -Endpoint <String> -Name <String> [-DefaultProfile <PSObject>] [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Deletes a data source

## EXAMPLES
