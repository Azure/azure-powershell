# Get-AzPurviewTrigger

## SYNOPSIS
Gets trigger information

## SYNTAX

```
Get-AzPurviewTrigger -Endpoint <String> -DataSourceName <String> -ScanName <String>
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Gets trigger information

## EXAMPLES
