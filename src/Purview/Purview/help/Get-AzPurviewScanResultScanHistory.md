# Get-AzPurviewScanResultScanHistory

## SYNOPSIS
Lists the scan history of a scan

## SYNTAX

```
Get-AzPurviewScanResultScanHistory -Endpoint <String> -DataSourceName <String> -ScanName <String>
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Lists the scan history of a scan
