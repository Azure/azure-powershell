# New-AzPurviewAmazonPostgreSqlScanRulesetObject

## SYNOPSIS
Create an in-memory object for AmazonPostgreSqlScanRuleset.

## SYNTAX

```
New-AzPurviewAmazonPostgreSqlScanRulesetObject -Kind <DataSourceType> [-Description <String>]
 [-ExcludedSystemClassification <String[]>] [-IncludedCustomClassificationRuleName <String[]>]
 [-Type <ScanRulesetType>] [<CommonParameters>]
```

## DESCRIPTION
Create an in-memory object for AmazonPostgreSqlScanRuleset.

## EXAMPLES
