# Get-AzConnectedVMwareVM

## SYNOPSIS
Retrieves information about a virtual machine instance.

## SYNTAX

### Get (Default)
```
Get-AzConnectedVMwareVM -MachineId <String> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### List
```
Get-AzConnectedVMwareVM -MachineId <String> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Retrieves information about a virtual machine instance.
