# Get-AzConnectedVMwareVMGuestAgent

## SYNOPSIS
Implements GuestAgent GET method.

## SYNTAX

```
Get-AzConnectedVMwareVMGuestAgent -MachineId <String> [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## DESCRIPTION
Implements GuestAgent GET method.

## EXAMPLES
