# Remove-AzCosmosDBSqlSoftDeletedDatabase

## SYNOPSIS
Permanently purges (removes) a soft-deleted SQL database from a CosmosDB account.

## SYNTAX

```
Remove-AzCosmosDBSqlSoftDeletedDatabase -ResourceGroupName <String> -AccountName <String> -Location <String>
 -Name <String> [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [-AcquirePolicyToken] [-ChangeReference <String>]
 [<CommonParameters>]
```

## DESCRIPTION
The **Remove-AzCosmosDBSqlSoftDeletedDatabase** cmdlet permanently purges a soft-deleted SQL database from a CosmosDB account. Once purged, the database cannot be recovered.

## EXAMPLES

### Example 1: Purge a soft-deleted SQL database
```powershell
Remove-AzCosmosDBSqlSoftDeletedDatabase -ResourceGroupName "myResourceGroup" -AccountName "myCosmosAccount" -Location "West US 2" -Name "myDeletedDatabase"
```

Permanently purges the soft-deleted SQL database named "myDeletedDatabase" from the specified CosmosDB account.
