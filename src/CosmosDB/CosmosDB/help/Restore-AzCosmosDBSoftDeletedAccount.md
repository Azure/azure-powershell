# Restore-AzCosmosDBSoftDeletedAccount

## SYNOPSIS
Restores a soft-deleted CosmosDB database account.

## SYNTAX

```
Restore-AzCosmosDBSoftDeletedAccount -ResourceGroupName <String> -Location <String> -Name <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [-AcquirePolicyToken] [-ChangeReference <String>] [<CommonParameters>]
```

## DESCRIPTION
The **Restore-AzCosmosDBSoftDeletedAccount** cmdlet restores a soft-deleted CosmosDB database account, making it accessible again.

## EXAMPLES

### Example 1: Restore a soft-deleted account
```powershell
Restore-AzCosmosDBSoftDeletedAccount -ResourceGroupName "myResourceGroup" -Location "West US 2" -Name "myDeletedAccount"
```

Restores the soft-deleted CosmosDB database account named "myDeletedAccount" in the specified resource group and location.
