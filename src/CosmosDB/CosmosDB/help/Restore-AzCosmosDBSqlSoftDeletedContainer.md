# Restore-AzCosmosDBSqlSoftDeletedContainer

## SYNOPSIS
Restores a soft-deleted SQL container in a CosmosDB SQL database.

## SYNTAX

```
Restore-AzCosmosDBSqlSoftDeletedContainer -ResourceGroupName <String> -AccountName <String> -Location <String>
 -DatabaseName <String> -Name <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [-AcquirePolicyToken] [-ChangeReference <String>]
 [<CommonParameters>]
```

## DESCRIPTION
The **Restore-AzCosmosDBSqlSoftDeletedContainer** cmdlet restores a soft-deleted SQL container in a CosmosDB SQL database, making it accessible again.

## EXAMPLES

### Example 1: Restore a soft-deleted SQL container
```powershell
Restore-AzCosmosDBSqlSoftDeletedContainer -ResourceGroupName "myResourceGroup" -AccountName "myCosmosAccount" -Location "West US 2" -DatabaseName "myDatabase" -Name "myDeletedContainer"
```

Restores the soft-deleted SQL container named "myDeletedContainer" in the specified database.
