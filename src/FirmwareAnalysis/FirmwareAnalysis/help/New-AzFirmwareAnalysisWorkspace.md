# New-AzFirmwareAnalysisWorkspace

## SYNOPSIS
The operation to Create a firmware analysis workspace.

## SYNTAX

```
New-AzFirmwareAnalysisWorkspace -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 -Location <String> [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Break]
 [-HttpPipelineAppend <SendAsyncStep[]>] [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <Uri>]
 [-ProxyCredential <PSCredential>] [-ProxyUseDefaultCredentials] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The operation to Create a firmware analysis workspace.

## EXAMPLES

### EXAMPLE 1
```
New-AzFirmwareAnalysisWorkspace -ResourceGroupName resourceGroupName -Name name -Location location
```
