# Restore-AzKeyVaultSecurityDomainBlob

## SYNOPSIS
Restores an exported managed HSM locally to get ready for importing.

## SYNTAX

```
Restore-AzKeyVaultSecurityDomainBlob -Keys <KeyPath[]> -ExchangeKey <String> -SecurityDomainPath <String>
 -SecurityDomainRestoreBlob <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [-SubscriptionId <String>] [<CommonParameters>]
```

## DESCRIPTION
This command will restore an exported managed HSM locally to get ready for importing.

## EXAMPLES

### Example 1
```powershell
$exchangeKey = "exchangeKey.cer"
$certsKeys = @{PublicKey = "sd1.cer"; PrivateKey = "sd1.key" }, @{PublicKey = "sd2.cer"; PrivateKey = "sd2.key" }, @{PublicKey = "sd3.cer"; PrivateKey = "sd3.key" }
Restore-AzKeyVaultSecurityDomainBlob -Keys $certsKeys -ExchangeKey $exchangeKey -SecurityDomainPath sd.json -SecurityDomainRestoreBlob restoreBlob.json
```

This command will decrypt the exported HSM stored in *sd.json* first, then encrypt it using exchange key generated by **Initialize-AzKeyVaultSecurityDomainRecovery**. Finally Store it in *restoreBlob.json*.
