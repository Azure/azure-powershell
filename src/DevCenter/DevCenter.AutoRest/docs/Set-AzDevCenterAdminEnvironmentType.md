# Set-AzDevCenterAdminEnvironmentType

## SYNOPSIS
Creates or updates an environment type.

## SYNTAX

### UpdateExpanded (Default)
```
Set-AzDevCenterAdminEnvironmentType -DevCenterName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```
