# Set-AzAdminProject

## SYNOPSIS
Creates or updates a project.

## SYNTAX

### UpdateExpanded (Default)
```
Set-AzAdminProject -Name <String> -ResourceGroupName <String> -Location <String> [-SubscriptionId <String>]
 [-Description <String>] [-DevCenterId <String>] [-MaxDevBoxesPerUser <Int32>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Update
```
Set-AzAdminProject -Name <String> -ResourceGroupName <String> -Body <IProject> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
