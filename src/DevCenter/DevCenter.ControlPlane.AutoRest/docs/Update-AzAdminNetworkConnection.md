# Update-AzAdminNetworkConnection

## SYNOPSIS
Partially updates a Network Connection

## SYNTAX

### UpdateExpanded (Default)
```
Update-AzAdminNetworkConnection -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DomainName <String>] [-DomainPassword <String>] [-DomainUsername <String>] [-Location <String>]
 [-OrganizationUnit <String>] [-SubnetId <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Update
```
Update-AzAdminNetworkConnection -Name <String> -ResourceGroupName <String> -Body <INetworkConnectionUpdate>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

```
Update-AzAdminNetworkConnection -InputObject <IDevCenterIdentity> -Body <INetworkConnectionUpdate>
```
