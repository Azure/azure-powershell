# Get-AzKustoSku

## SYNOPSIS
Lists eligible region SKUs for Kusto resource provider by Azure region.

## SYNTAX

```
Get-AzKustoSku -Location <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## DESCRIPTION
Lists eligible region SKUs for Kusto resource provider by Azure region.
