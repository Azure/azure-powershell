# Get-AzSynapseSqlTransparentDataEncryptionProtector

## SYNOPSIS
Gets the Transparent Data Encryption (TDE) protector.

## SYNTAX

### GetByNameParameterSet (Default)
```
Get-AzSynapseSqlTransparentDataEncryptionProtector [-ResourceGroupName <String>] -WorkspaceName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByInputObjectParameterSet
```
Get-AzSynapseSqlTransparentDataEncryptionProtector -InputObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseSqlTransparentDataEncryptionProtector -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
The **Get-AzSynapseSqlTransparentDataEncryptionProtector** cmdlet gets information about the TDE protector.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-AzSynapseSqlTransparentDataEncryptionProtector -WorkspaceName ContosoWorkspace
```

This command gets the TDE protector for the workspace named ContosoWorkspace.
