# New-AzMigratePolicy

## SYNOPSIS
Create an environment in the specified subscription and resource group.

## SYNTAX

```
New-AzMigratePolicy -PolicyName <String> -ResourceGroupName <String> -ResourceName <String>
 -AppConsistentFrequencyInMinutes <Int32> -CrashConsistentFrequencyInMinutes <Int32>
 -ProviderSpecificInputInstanceType <String> -RecoveryPointHistoryInMinutes <Int32> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
