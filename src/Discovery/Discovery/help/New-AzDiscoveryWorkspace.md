# New-AzDiscoveryWorkspace

## SYNOPSIS
Create a Workspace

## SYNTAX

### CreateExpanded (Default)
```
New-AzDiscoveryWorkspace -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 -Location <String> [-AgentSubnetId <String>] [-CustomerManagedKey <String>]
 [-KeyVaultPropertyKeyName <String>] [-KeyVaultPropertyKeyVaultUri <String>]
 [-KeyVaultPropertyKeyVersion <String>] [-LogAnalyticsClusterId <String>] [-PrivateEndpointSubnetId <String>]
 [-PublicNetworkAccess <String>] [-SupercomputerId <String[]>] [-Tag <Hashtable>]
 [-WorkspaceIdentityId <String>] [-WorkspaceSubnetId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### CreateViaJsonFilePath
```
New-AzDiscoveryWorkspace -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 -JsonFilePath <String> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### CreateViaJsonString
```
New-AzDiscoveryWorkspace -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 -JsonString <String> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Create a Workspace

## EXAMPLES

### Example 1: Create a new workspace
```powershell
New-AzDiscoveryWorkspace -ResourceGroupName "my-rg" -Name "my-workspace" -Location "eastus" -SupercomputerId @("/subscriptions/{subId}/resourceGroups/my-rg/providers/Microsoft.Discovery/supercomputers/my-supercomputer")
```

```output
Location    Name            ResourceGroupName    ProvisioningState
--------    ----            -----------------    -----------------
eastus      my-workspace    my-rg                Succeeded
```

Creates a new Discovery workspace associated with a supercomputer.
