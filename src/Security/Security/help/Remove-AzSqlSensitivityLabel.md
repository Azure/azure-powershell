# Remove-AzSqlSensitivityLabel

## SYNOPSIS
Removes a sensitivity label from the SQL information protection policy.

## SYNTAX

```
Remove-AzSqlSensitivityLabel [-PassThru] -DisplayName <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Removes a sensitivity label from the SQL information protection policy.

## EXAMPLES

### Example
```powershell
Remove-AzSqlSensitivityLabel -DisplayName General
```
