# Set-AzureRmExpressRouteCircuit

## SYNOPSIS

```

 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRIPTION
The **Set-AzureRmExpressRouteCircuit** cmdlet modifies an ExpressRoute circuit.
