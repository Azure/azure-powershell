# New-AzureRmManagementGroupSubscription

## SYNOPSIS
Adds a Subscription to a Management Group.

## SYNTAX

```
New-AzureRmManagementGroupSubscription [-GroupName] <String> [-SubscriptionId] <Guid> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
```

## DESCRIPTION
The **New-AzureRMManagementGroupSubscription** cmdlet adds a Subscription to a Management Group.

## EXAMPLES

### Example 1: Add Subscription to a Management Group
```
PS C:\> Add-AzureRMManagementGroupSubscription -GroupName "TestGroup" -SubscriptionId 2120692d-35c3-44c8-81f5-631fa7351726
```
