# Get-AzureRmRelayOperation

## SYNOPSIS
List supported Relay Operations

## SYNTAX

```
Get-AzureRmRelayOperation
```

## DESCRIPTION
The **Get-AzureRmRelayOperation** cmdlet Lists the Relay supported Operations.

## EXAMPLES

### Example 1
```
PS C:\> Get-AzureRmRelayOperation
```

Lists Relay supported operations
