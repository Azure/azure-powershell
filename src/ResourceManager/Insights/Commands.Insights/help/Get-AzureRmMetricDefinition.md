# Get-AzureRmMetricDefinition

## SYNOPSIS
Gets metric definitions.

## SYNTAX

```
Get-AzureRmMetricDefinition [-ResourceId] <String> [-MetricNames <String[]>] [-DetailedOutput]
 [<CommonParameters>]
```

## DESCRIPTION
The **Get-AzureRmMetricDefinition** cmdlet gets metric definitions.
