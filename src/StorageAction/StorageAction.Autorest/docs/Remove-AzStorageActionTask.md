# Remove-AzStorageActionTask

## SYNOPSIS
Delete the storage task resource.

## SYNTAX

### Delete (Default)
```
Remove-AzStorageActionTask -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzStorageActionTask -InputObject <IStorageActionIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Delete the storage task resource.

## EXAMPLES

### Example 1: Delete specific storage action task with specified resource group
```powershell
Remove-AzStorageActionTask -Name mytask1 -ResourceGroupName group001
```

This command delete specific storage action task with specified resource group
