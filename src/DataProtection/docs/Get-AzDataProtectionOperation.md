# Get-AzDataProtectionOperation

## SYNOPSIS
Returns the list of available operations.

## SYNTAX

```
Get-AzDataProtectionOperation [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Returns the list of available operations.
