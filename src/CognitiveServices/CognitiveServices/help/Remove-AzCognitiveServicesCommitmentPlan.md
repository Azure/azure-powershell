# Remove-AzCognitiveServicesCommitmentPlan

## SYNOPSIS
Remove a Cognitive Services Commitment Plan

## SYNTAX

```
Remove-AzCognitiveServicesCommitmentPlan [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Remove a Cognitive Services Commitment Plan

## EXAMPLES

### Example 1
```powershell
Remove-AzCognitiveServicesCommitmentPlan -ResourceGroupName ResourceGroupName -Name CommitmentPlanName
```

Remove a Cognitive Services Commitment Plan
