# Remove-AzElasticDetachAndDeleteTrafficFilter

## SYNOPSIS
Detach and Delete traffic filter from the given deployment.

## SYNTAX

### Delete (Default)
```
Remove-AzElasticDetachAndDeleteTrafficFilter -MonitorName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-RulesetId <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzElasticDetachAndDeleteTrafficFilter -InputObject <IElasticIdentity> [-RulesetId <String>]
 [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Detach and Delete traffic filter from the given deployment.

## EXAMPLES

### Example 1: Detach and Delete traffic filter from the given deployment.
```powershell
Remove-AzElasticDetachAndDeleteTrafficFilter -ResourceGroupName azure-elastic-test -MonitorName elastic-pwsh02
```

This command will detach and delete the traffic filter.
