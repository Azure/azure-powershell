# Invoke-AzNetworkFabricTapUpdateAdminState

## SYNOPSIS
Implements the operation to the underlying resources.

## SYNTAX

### UpdateExpanded (Default)
```
Invoke-AzNetworkFabricTapUpdateAdminState -NetworkTapName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-ResourceId <String[]>] [-State <String>] [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

```

 [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Implements the operation to the underlying resources.

## EXAMPLES

### Example 1: Enable/Disable a Network Tap
```powershell
$state="Enable"
Invoke-AzNetworkFabricTapUpdateAdminState -NetworkTapName $name -ResourceGroupName $resourceGroupName -State $state
```

This command Enable/Disable a network tap
