# New-AzNetworkCloudRack

## SYNOPSIS
Create a new rack or update properties of the existing one.\nAll customer initiated requests will be rejected as the life cycle of this resource is managed by the system.

## SYNTAX

```
New-AzNetworkCloudRack -Name <String> -ResourceGroupName <String> -AvailabilityZone <String>
 -ExtendedLocationName <String> -ExtendedLocationType <String> -Location <String> -RackLocation <String>
 -RackSerialNumber <String> -RackSkuId <String> [-SubscriptionId <String>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create a new rack or update properties of the existing one.\nAll customer initiated requests will be rejected as the life cycle of this resource is managed by the system.
