# Invoke-AzNetworkCloudBareMetalMachineUncordon

## SYNOPSIS
Uncordon the provided bare metal machine's Kubernetes node.

## SYNTAX

### Uncordon (Default)
```
Invoke-AzNetworkCloudBareMetalMachineUncordon -BareMetalMachineName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UncordonViaIdentity
```
Invoke-AzNetworkCloudBareMetalMachineUncordon -InputObject <INetworkCloudIdentity> [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-PassThru] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Uncordon the provided bare metal machine's Kubernetes node.

## EXAMPLES

### Example 1: Uncordon a bare metal machine
```powershell
Invoke-AzNetworkCloudBareMetalMachineUncordon -BareMetalMachineName bmmName -ResourceGroupName resourceGroupName -SubscriptionId subscriptionId
```

This command uncordons a bare metal machine.
