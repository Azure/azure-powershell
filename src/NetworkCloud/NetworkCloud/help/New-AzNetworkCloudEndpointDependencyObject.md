# New-AzNetworkCloudEndpointDependencyObject

## SYNOPSIS

## SYNTAX

```
New-AzNetworkCloudEndpointDependencyObject -DomainName <String> [-Port <Int64>]
 [<CommonParameters>]
```

## DESCRIPTION
Create an in-memory object for EndpointDependency.
