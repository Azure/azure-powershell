# New-AzCertificate

## SYNOPSIS
Create a Certificate to the specified Account.

## SYNTAX

### CreateExpanded (Default)
```
New-AzCertificate -Endpoint <String> -DataInputFile <String> -Thumbprint <String>
 -ThumbprintAlgorithm <String> [-TimeOut <Int32>] [-ClientRequestId <String>] [-Ocpdate <String>]
 [-ReturnClientRequestId] [-CertificateFormat <String>] [-Password <String>] [-DefaultProfile <PSObject>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaJsonFilePath
```
New-AzCertificate -Endpoint <String> -JsonFilePath <String> [-TimeOut <Int32>] [-ClientRequestId <String>]
 [-Ocpdate <String>] [-ReturnClientRequestId] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### CreateViaJsonString
```
New-AzCertificate -Endpoint <String> -JsonString <String> [-TimeOut <Int32>] [-ClientRequestId <String>]
 [-Ocpdate <String>] [-ReturnClientRequestId] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Create a Certificate to the specified Account.
