# Get-AzLegacyPeering

## SYNOPSIS
Used to Convert Legacy Peering resources to Azure Resource Management (ARM) Resources. 

## SYNTAX

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
The command is used to view legacy Peering resources which all you to convert them to Azure Resource Management (ARM) Resources.
