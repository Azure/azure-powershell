# Open-AzSurveyLink

## SYNOPSIS
Open survey link in default browser.

## SYNTAX

```
Open-AzSurveyLink [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Open survey link in default browser.

## EXAMPLES

### Example 1
```
Open-AzSurveyLink
Opening the default browser to https://aka.ms/azpssurvey?Q_CHL=INTERCEPT
```
