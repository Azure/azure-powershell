# Get-AzPureStorageBlockReservationBillingReport

## SYNOPSIS
Provides a summarized report along with actions for resources billed via given reservation

## SYNTAX

### Get (Default)
```
Get-AzPureStorageBlockReservationBillingReport -ReservationName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzPureStorageBlockReservationBillingReport -InputObject <IPureStorageBlockIdentity>
 [-DefaultProfile <PSObject>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Provides a summarized report along with actions for resources billed via given reservation
