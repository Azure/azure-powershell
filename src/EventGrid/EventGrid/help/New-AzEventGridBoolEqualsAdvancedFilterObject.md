# New-AzEventGridBoolEqualsAdvancedFilterObject

## SYNOPSIS
Create an in-memory object for BoolEqualsAdvancedFilter.

## SYNTAX

```
New-AzEventGridBoolEqualsAdvancedFilterObject [-Value <Boolean>] [-Key <String>]
 [<CommonParameters>]
```

## DESCRIPTION
