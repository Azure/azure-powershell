# Get-AzWebAppMetric

## SYNOPSIS
Gets performance metrics of an app (or deployment slot, if specified).

## SYNTAX

```
Get-AzWebAppMetric -Name <String> -ResourceGroupName <String> -SubscriptionId <String[]> [-Detail <Boolean>]
 [-Filter <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Gets performance metrics of an app (or deployment slot, if specified).
