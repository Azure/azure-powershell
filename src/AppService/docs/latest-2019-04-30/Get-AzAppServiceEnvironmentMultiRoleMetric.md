# Get-AzAppServiceEnvironmentMultiRoleMetric

## SYNOPSIS
Get metrics for a multi-role pool of an App Service Environment.

## SYNTAX

```
Get-AzAppServiceEnvironmentMultiRoleMetric -Name <String> -ResourceGroupName <String>
 -SubscriptionId <String[]> [-Detail] [-EndTime <String>] [-Filter <String>] [-StartTime <String>]
 [-TimeGrain <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Get metrics for a multi-role pool of an App Service Environment.
