# Remove-AzWebAppProcessSlot

## SYNOPSIS
Terminate a process by its ID for a web site, or a deployment slot, or specific scaled-out instance in a web site.

## SYNTAX

### Delete (Default)
```
Remove-AzWebAppProcessSlot -Name <String> -ProcessId <String> -ResourceGroupName <String> -Slot <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzWebAppProcessSlot -InputObject <IAppServiceIdentity> [-DefaultProfile <PSObject>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Terminate a process by its ID for a web site, or a deployment slot, or specific scaled-out instance in a web site.
