# Set-AzMySqlBackup

## SYNOPSIS
Create backup for a given server with specified backup name.

## SYNTAX

```
Set-AzMySqlBackup -Name <String> -ResourceGroupName <String> -ServerName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create backup for a given server with specified backup name.
