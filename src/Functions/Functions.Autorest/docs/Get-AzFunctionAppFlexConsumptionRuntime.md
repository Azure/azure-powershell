## SYNOPSIS

## SYNTAX

```
Get-AzFunctionAppFlexConsumptionRuntime -Location <String> -Runtime <String> [-Version <String>]
 [<CommonParameters>]
```

## DESCRIPTION
Gets the Flex Consumption function app runtimes supported at the specified location.
