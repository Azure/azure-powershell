# Remove-AzChaosScenario

## SYNOPSIS
Delete a scenario.

## SYNTAX

### Delete (Default)
```
Remove-AzChaosScenario -Name <String> -ResourceGroupName <String> -WorkspaceName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzChaosScenario -InputObject <IChaosIdentity> [-DefaultProfile <PSObject>] [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentityWorkspace
```
Remove-AzChaosScenario -Name <String> -WorkspaceInputObject <IChaosIdentity> [-DefaultProfile <PSObject>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Delete a scenario.
