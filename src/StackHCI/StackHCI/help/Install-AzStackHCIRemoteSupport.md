# Install-AzStackHCIRemoteSupport

## SYNOPSIS
Installs Remote Support.

## SYNTAX

```
Install-AzStackHCIRemoteSupport [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Install-AzStackHCIRemoteSupport installs Remote Support Deployment module.
