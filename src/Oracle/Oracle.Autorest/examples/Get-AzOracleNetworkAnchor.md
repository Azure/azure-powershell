### Example 1: Get Network Anchors in a resource group
```powershell
Get-AzOracleNetworkAnchor -ResourceGroupName PowerShellTestRg
```

Lists the Network Anchors in the specified resource group. For more information, execute `Get-Help Get-AzOracleNetworkAnchor`.

### Example 2: Get a Network Anchor by name
```powershell
Get-AzOracleNetworkAnchor -ResourceGroupName PowerShellTestRg -Name PowerShellTestNetworkAnchor
```

Gets the specified Network Anchor. For more information, execute `Get-Help Get-AzOracleNetworkAnchor`.
