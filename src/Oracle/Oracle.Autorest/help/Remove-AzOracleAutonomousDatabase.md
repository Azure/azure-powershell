# Remove-AzOracleAutonomousDatabase

## SYNOPSIS
Delete a AutonomousDatabase

## SYNTAX

### Delete (Default)
```
Remove-AzOracleAutonomousDatabase -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzOracleAutonomousDatabase -InputObject <IOracleIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Delete a AutonomousDatabase

## EXAMPLES

### Example 1: Delete an Autonomous Database resource

Remove-AzOracleAutonomousDatabase -NoWait -Name "OFakePowerShellTestAdbs" -ResourceGroupName "PowerShellTestRg"
