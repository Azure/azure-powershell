# Open-AzSurveyLink

## SYNOPSIS
Open survey link in default browser.

## SYNTAX

```
Open-AzSurveyLink [<CommonParameters>]
```

## DESCRIPTION
Open survey link in default browser.

## EXAMPLES

### Example 1
```powershell
Open-AzSurveyLink
```

```Output
Opening the default browser to https://aka.ms/azpssurvey?Q_CHL=INTERCEPT
```
