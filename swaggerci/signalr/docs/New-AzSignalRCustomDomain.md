## SYNOPSIS

```
 -DomainName <String> [-SubscriptionId <String>] [-CustomCertificateId <String>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
