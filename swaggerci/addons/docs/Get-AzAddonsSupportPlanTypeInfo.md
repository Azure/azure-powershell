```
Get-AzAddonsSupportPlanTypeInfo [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Returns the canonical support plan information for all types for the subscription.
