# Get-AzMachineLearningComputeOperationalizationClusterKey

## SYNOPSIS
Gets the credentials for the specified cluster such as Storage, ACR and ACS credentials.
This is a long running operation because it fetches keys from dependencies.

## SYNTAX

```
Get-AzMachineLearningComputeOperationalizationClusterKey -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Gets the credentials for the specified cluster such as Storage, ACR and ACS credentials.
This is a long running operation because it fetches keys from dependencies.
