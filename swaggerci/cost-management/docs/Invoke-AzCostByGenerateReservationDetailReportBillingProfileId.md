# Invoke-AzCostByGenerateReservationDetailReportBillingProfileId

## SYNOPSIS
Generates the reservations details report for provided date range asynchronously by billing profile.

## SYNTAX

### By (Default)
```
Invoke-AzCostByGenerateReservationDetailReportBillingProfileId -BillingAccountId <String>
 -BillingProfileId <String> -EndDate <String> -StartDate <String> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### ByViaIdentity
```
Invoke-AzCostByGenerateReservationDetailReportBillingProfileId -InputObject <ICostIdentity> -EndDate <String>
 -StartDate <String> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Generates the reservations details report for provided date range asynchronously by billing profile.
