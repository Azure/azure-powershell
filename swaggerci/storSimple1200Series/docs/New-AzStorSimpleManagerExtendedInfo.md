# New-AzStorSimpleManagerExtendedInfo

## SYNOPSIS
Creates the extended info of the manager.

## SYNTAX

```
New-AzStorSimpleManagerExtendedInfo -ManagerName <String> -ResourceGroupName <String> -Algorithm <String>
 -IntegrityKey <String> [-SubscriptionId <String>] [-EncryptionKey <String>]
 [-EncryptionKeyThumbprint <String>] [-Etag <String>] [-PortalCertificateThumbprint <String>]
 [-Version <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates the extended info of the manager.
