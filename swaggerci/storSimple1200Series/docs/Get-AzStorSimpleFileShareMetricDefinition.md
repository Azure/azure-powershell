# Get-AzStorSimpleFileShareMetricDefinition

## SYNOPSIS
Retrieves metric definitions of all metrics aggregated at the file share.

## SYNTAX

```
Get-AzStorSimpleFileShareMetricDefinition -DeviceName <String> -FileServerName <String> -ManagerName <String>
 -ResourceGroupName <String> -ShareName <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## DESCRIPTION
Retrieves metric definitions of all metrics aggregated at the file share.
