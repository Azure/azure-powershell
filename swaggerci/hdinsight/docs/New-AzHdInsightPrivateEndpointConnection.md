# New-AzHdInsightPrivateEndpointConnection

## SYNOPSIS
Approve or reject a private endpoint connection manually.

## SYNTAX

```
New-AzHdInsightPrivateEndpointConnection -ClusterName <String> -Name <String> -ResourceGroupName <String>
 -PrivateLinkServiceConnectionStateStatus <PrivateLinkServiceConnectionStatus> [-SubscriptionId <String>]
 [-PrivateLinkServiceConnectionStateActionsRequired <String>]
 [-PrivateLinkServiceConnectionStateDescription <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Approve or reject a private endpoint connection manually.
