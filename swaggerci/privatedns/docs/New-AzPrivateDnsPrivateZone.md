# New-AzPrivateDnsPrivateZone

## SYNOPSIS
Creates or updates a Private DNS zone.
Does not modify Links to virtual networks or DNS records within the zone.

## SYNTAX

```
New-AzPrivateDnsPrivateZone -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-IfMatch <String>] [-IfNoneMatch <String>] [-Etag <String>] [-Location <String>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates or updates a Private DNS zone.
Does not modify Links to virtual networks or DNS records within the zone.
