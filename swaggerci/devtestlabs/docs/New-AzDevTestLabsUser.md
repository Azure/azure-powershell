# New-AzDevTestLabsUser

## SYNOPSIS
Create or replace an existing user profile.
This operation can take a while to complete.

## SYNTAX

```
New-AzDevTestLabsUser -LabName <String> -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-IdentityAppId <String>] [-IdentityObjectId <String>] [-IdentityPrincipalId <String>]
 [-IdentityPrincipalName <String>] [-IdentityTenantId <String>] [-Location <String>]
 [-SecretStoreKeyVaultId <String>] [-SecretStoreKeyVaultUri <String>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create or replace an existing user profile.
This operation can take a while to complete.
