# New-AzDevTestLabsServiceFabric

## SYNOPSIS
Create or replace an existing service fabric.
This operation can take a while to complete.

## SYNTAX

```
New-AzDevTestLabsServiceFabric -LabName <String> -Name <String> -ResourceGroupName <String> -UserName <String>
 [-SubscriptionId <String>] [-EnvironmentId <String>] [-ExternalServiceFabricId <String>] [-Location <String>]
 [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
