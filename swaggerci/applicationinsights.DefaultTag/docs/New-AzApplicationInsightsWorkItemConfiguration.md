# New-AzApplicationInsightsWorkItemConfiguration

## SYNOPSIS

## SYNTAX

```
New-AzApplicationInsightsWorkItemConfiguration -ResourceGroupName <String> -ResourceName <String>
 [-SubscriptionId <String>] [-ConnectorDataConfiguration <String>] [-ConnectorId <String>] [-ValidateOnly]
 [-WorkItemProperty <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

Create a work item configuration for an Application Insights component.
