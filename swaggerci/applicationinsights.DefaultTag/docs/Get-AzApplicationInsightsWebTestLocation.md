# Get-AzApplicationInsightsWebTestLocation

## SYNOPSIS
Gets a list of web test locations available to this Application Insights component.

## SYNTAX

```
Get-AzApplicationInsightsWebTestLocation -ResourceGroupName <String> -ResourceName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Gets a list of web test locations available to this Application Insights component.
