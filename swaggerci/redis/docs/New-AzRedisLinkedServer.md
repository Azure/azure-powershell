# New-AzRedisLinkedServer

## SYNOPSIS
Adds a linked server to the Redis cache (requires Premium SKU).

## SYNTAX

```
New-AzRedisLinkedServer -LinkedServerName <String> -Name <String> -ResourceGroupName <String>
 -LinkedRedisCacheId <String> -LinkedRedisCacheLocation <String> -ServerRole <ReplicationRole>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Adds a linked server to the Redis cache (requires Premium SKU).
