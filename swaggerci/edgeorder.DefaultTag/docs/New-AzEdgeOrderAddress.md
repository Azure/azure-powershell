# New-AzEdgeOrderAddress

## SYNOPSIS
Create a new address with the specified parameters.
Existing address cannot be updated with this API and should\r\ninstead be updated with the Update address API.

## SYNTAX

```
New-AzEdgeOrderAddress -Name <String> -ResourceGroupName <String> -ContactDetailContactName <String>
 -ContactDetailEmailList <String[]> -ContactDetailPhone <String> -Location <String> [-SubscriptionId <String>]
 [-ContactDetailMobile <String>] [-ContactDetailPhoneExtension <String>] [-ShippingAddressCity <String>]
 [-ShippingAddressCompanyName <String>] [-ShippingAddressCountry <String>]
 [-ShippingAddressPostalCode <String>] [-ShippingAddressStateOrProvince <String>]
 [-ShippingAddressStreetAddress1 <String>] [-ShippingAddressStreetAddress2 <String>]
 [-ShippingAddressStreetAddress3 <String>] [-ShippingAddressType <AddressType>]
 [-ShippingAddressZipExtendedCode <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create a new address with the specified parameters.
Existing address cannot be updated with this API and should\r\ninstead be updated with the Update address API.
