```
Update-AzSaaSSaaSubscriptionLevelToUnsubscribed -ResourceGroupName <String> -ResourceName <String>
 [-SubscriptionId <String>] [-Feedback <String>] [-ReasonCode <Single>] [-UnsubscribeOnly]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
