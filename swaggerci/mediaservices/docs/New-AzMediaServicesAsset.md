## SYNOPSIS

```
 [-SubscriptionId <String>] [-AlternateId <String>] [-Container <String>] [-Description <String>]
 [-StorageAccountName <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```
