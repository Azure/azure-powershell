# Get-AzDataLakeAnalyticsAccountStorageAccountSasToken

## SYNOPSIS
Gets the SAS token associated with the specified Data Lake Analytics and Azure Storage account and container combination.

## SYNTAX

```
Get-AzDataLakeAnalyticsAccountStorageAccountSasToken -AccountName <String> -ContainerName <String>
 -ResourceGroupName <String> -StorageAccountName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Gets the SAS token associated with the specified Data Lake Analytics and Azure Storage account and container combination.
