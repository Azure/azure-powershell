# New-AzDataBoxEdgeStorageAccount

## SYNOPSIS
Creates a new StorageAccount or updates an existing StorageAccount on the device.

## SYNTAX

```
New-AzDataBoxEdgeStorageAccount -DeviceName <String> -Name <String> -ResourceGroupName <String>
 -DataPolicy <DataPolicy> [-SubscriptionId <String>] [-Description <String>]
 [-StorageAccountCredentialId <String>] [-StorageAccountStatus <StorageAccountStatus>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates a new StorageAccount or updates an existing StorageAccount on the device.
