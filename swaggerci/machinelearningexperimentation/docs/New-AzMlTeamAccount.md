# New-AzMlTeamAccount

## SYNOPSIS
Creates or updates a team account with the specified parameters.

## SYNTAX

```

 [-Description <String>] [-FriendlyName <String>] [-KeyVaultId <String>] [-Seat <String>]
 [-StorageAccountAccessKey <String>] [-StorageAccountId <String>] [-Tag <Hashtable>] [-VsoAccountId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```
