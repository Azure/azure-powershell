# New-AzAutomationWebhook

## SYNOPSIS
Create the webhook identified by webhook name.

## SYNTAX

```
New-AzAutomationWebhook -AutomationAccountName <String> -ResourceGroupName <String> -WebhookName <String>
 -Name <String> [-SubscriptionId <String>] [-ExpiryTime <DateTime>] [-IsEnabled] [-Parameter <Hashtable>]
 [-RunbookName <String>] [-RunOn <String>] [-Uri <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Create the webhook identified by webhook name.
