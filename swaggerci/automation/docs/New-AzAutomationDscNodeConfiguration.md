# New-AzAutomationDscNodeConfiguration

## SYNOPSIS
Create the node configuration identified by node configuration name.

## SYNTAX

```
New-AzAutomationDscNodeConfiguration -AutomationAccountName <String> -NodeConfigurationName <String>
 -ResourceGroupName <String> [-SubscriptionId <String>] [-ConfigurationName <String>]
 [-HashAlgorithm <String>] [-HashValue <String>] [-IncrementNodeConfigurationBuild] [-Name <String>]
 [-SourceType <ContentSourceType>] [-SourceValue <String>] [-SourceVersion <String>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create the node configuration identified by node configuration name.
