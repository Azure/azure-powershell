# New-AzAutomationCertificate

## SYNOPSIS

## SYNTAX

```

 -ResourceGroupName <String> -Base64Value <String> -Name <String> [-SubscriptionId <String>]
 [-Description <String>] [-IsExportable] [-Thumbprint <String>] [-DefaultProfile <PSObject>] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```
