# New-AzAutomationSourceControlSyncJob

## SYNOPSIS
Creates the sync job for a source control.

## SYNTAX

```
New-AzAutomationSourceControlSyncJob -AutomationAccountName <String> -Id <String> -ResourceGroupName <String>
 -SourceControlName <String> -CommitId <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Creates the sync job for a source control.
