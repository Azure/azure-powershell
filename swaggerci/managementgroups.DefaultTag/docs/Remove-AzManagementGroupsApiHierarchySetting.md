# Remove-AzManagementGroupsApiHierarchySetting

## SYNOPSIS
Deletes the hierarchy settings defined at the Management Group level.\n

## SYNTAX

### Delete (Default)
```
Remove-AzManagementGroupsApiHierarchySetting -GroupId <String> [-DefaultProfile <PSObject>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzManagementGroupsApiHierarchySetting -InputObject <IManagementGroupsApiIdentity>
 [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Deletes the hierarchy settings defined at the Management Group level.\n
