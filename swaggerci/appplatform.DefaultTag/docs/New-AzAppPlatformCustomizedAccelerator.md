# New-AzAppPlatformCustomizedAccelerator

## SYNOPSIS
Create or update the customized accelerator.

## SYNTAX

```
New-AzAppPlatformCustomizedAccelerator -ApplicationAcceleratorName <String> -Name <String>
 -ResourceGroupName <String> -ServiceName <String> [-SubscriptionId <String>] [-AcceleratorTag <String[]>]
 [-AuthSettingAuthType <String>] [-Description <String>] [-DisplayName <String>]
 [-GitRepositoryBranch <String>] [-GitRepositoryCommit <String>] [-GitRepositoryGitTag <String>]
 [-GitRepositoryIntervalInSecond <Int32>] [-GitRepositoryUrl <String>] [-IconUrl <String>]
 [-SkuCapacity <Int32>] [-SkuName <String>] [-SkuTier <String>] [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
