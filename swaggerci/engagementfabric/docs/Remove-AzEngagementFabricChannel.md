# Remove-AzEngagementFabricChannel

## SYNOPSIS
Delete the EngagementFabric channel

## SYNTAX

### Delete (Default)
```
Remove-AzEngagementFabricChannel -AccountName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzEngagementFabricChannel -InputObject <IEngagementFabricIdentity> [-DefaultProfile <PSObject>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
