# New-AzTimeSeriesInsightsEnvironment

## SYNOPSIS
Create or update an environment in the specified subscription and resource group.

## SYNTAX

```
New-AzTimeSeriesInsightsEnvironment -Name <String> -ResourceGroupName <String> -Kind <EnvironmentKind>
 -Location <String> -SkuCapacity <Int32> -SkuName <SkuName> [-SubscriptionId <String>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
