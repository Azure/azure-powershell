# New-AzDevOpsPipeline

## SYNOPSIS
Creates or updates an Azure Pipeline.

 -ProjectName <String> -TemplateId <String> [-SubscriptionId <String>] [-AuthorizationParameter <Hashtable>]
 [-Location <String>] [-RepositoryDefaultBranch <String>] [-RepositoryId <String>]
 [-RepositoryProperty <Hashtable>] [-RepositoryType <CodeRepositoryType>] [-Tag <Hashtable>]
 [-TemplateParameter <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
