# Invoke-AzMachineLearningWorkspacesPrepareWorkspaceNotebook

## SYNOPSIS
Prepare a notebook.

## SYNTAX

### Prepare (Default)
```
Invoke-AzMachineLearningWorkspacesPrepareWorkspaceNotebook -ResourceGroupName <String> -WorkspaceName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

