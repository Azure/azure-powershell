# New-AzDataLakeStoreAccountVirtualNetworkRule

## SYNOPSIS
Creates or updates the specified virtual network rule.
During update, the virtual network rule with the specified name will be replaced with this new virtual network rule.

## SYNTAX

```
New-AzDataLakeStoreAccountVirtualNetworkRule -AccountName <String> -Name <String> -ResourceGroupName <String>
 -SubnetId <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Creates or updates the specified virtual network rule.
During update, the virtual network rule with the specified name will be replaced with this new virtual network rule.
