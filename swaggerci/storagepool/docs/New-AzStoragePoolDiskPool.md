# New-AzStoragePoolDiskPool

## SYNOPSIS
Create or Update Disk pool.
This create or update operation can take 15 minutes to complete.
This is expected service behavior.

## SYNTAX

```
New-AzStoragePoolDiskPool -Name <String> -ResourceGroupName <String> -Location <String> -SkuName <String>
 -SubnetId <String> [-SubscriptionId <String>] [-AdditionalCapability <String[]>]
 [-AvailabilityZone <String[]>] [-Disk <IDisk[]>] [-ManagedBy <String>] [-ManagedByExtended <String[]>]
 [-SkuTier <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Create or Update Disk pool.
