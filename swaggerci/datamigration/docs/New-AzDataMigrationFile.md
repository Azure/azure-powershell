# New-AzDataMigrationFile

## SYNOPSIS
The PUT method creates a new file or updates an existing one.

## SYNTAX

```
 [-SubscriptionId <String>] [-Etag <String>] [-Extension <String>] [-FilePath <String>] [-MediaType <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
The PUT method creates a new file or updates an existing one.
