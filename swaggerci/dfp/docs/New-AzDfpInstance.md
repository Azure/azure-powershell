# New-AzDfpInstance

## SYNOPSIS
Provisions the specified DFP instance based on the configuration specified in the request.

## SYNTAX

```
New-AzDfpInstance -Name <String> -ResourceGroupName <String> -Location <String> [-SubscriptionId <String>]
 [-AdministrationMember <String[]>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
