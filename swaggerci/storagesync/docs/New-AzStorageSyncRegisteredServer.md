## SYNOPSIS

```
 [-ClusterName <String>] [-FriendlyName <String>] [-LastHeartBeat <String>] [-PropertiesServerId <String>]
 [-ServerCertificate <String>] [-ServerOSVersion <String>] [-ServerRole <String>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```
