# Remove-AzElasticTagRule

## SYNOPSIS
Delete a tag rule set for a given monitor resource.

## SYNTAX

### Delete (Default)
```
Remove-AzElasticTagRule -MonitorName <String> -ResourceGroupName <String> -RuleSetName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzElasticTagRule -InputObject <IElasticIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Delete a tag rule set for a given monitor resource.
