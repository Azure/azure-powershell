# Get-AzArcSqlManagedInstance

## SYNOPSIS
Retrieves a SQL Managed Instance resource

## SYNTAX

### List (Default)
```
Get-AzArcSqlManagedInstance [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Get
```
Get-AzArcSqlManagedInstance -Name <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
```
