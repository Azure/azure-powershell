# New-AzArcPostgresInstance

## SYNOPSIS
Creates or replaces a postgres Instance resource

## SYNTAX

```
New-AzArcPostgresInstance -Name <String> -ResourceGroupName <String> -Location <String>
 [-SubscriptionId <String>] [-Admin <String>] [-BasicLoginInformationPassword <String>]
 [-BasicLoginInformationUsername <String>] [-DataControllerId <String>] [-ExtendedLocationName <String>]
 [-ExtendedLocationType <ExtendedLocationTypes>] [-K8SRaw <IAny>] [-LastUploadedDate <DateTime>]
 [-SkuCapacity <Int32>] [-SkuDev] [-SkuFamily <String>] [-SkuName <String>] [-SkuSize <String>]
 [-SkuTier <PostgresInstanceSkuTier>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```
